# 3D MacBook Showcase

A modern, interactive 3D landing page built with **Three.js** and **GSAP**, designed to replicate the smooth animations and premium feel of Apple-style product pages. This project focuses on high-quality motion, performance optimization, and a clean, professional user experience.
